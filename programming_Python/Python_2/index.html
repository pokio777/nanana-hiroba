2. -2- Pythonノック1から5の解説

2.1 問題に出題された知識の確認

Pythonのプログラムは、コードの上から下に向かって順番に実行されます。また、重要なのが「インデント」です。

文章の行頭に空白を挿入することをインデントと呼びますが、Pythonでは、基本的に「半角スペース4個分」を1つのコードのまとまり（ブロック）として扱います。これを正しく行わないと、コンピュータがコードの繋がりを理解できずエラーになってしまいます。

今回のメインテーマは「型（かた）」です。Pythonにおける型とは、一言で説明すると「データの種類と、その振る舞いのルール」です。

例えば「10」というデータがあるとき、それが数値（計算に使うもの）なのか、それとも文字（そのまま表示するもの）なのかをコンピュータに教える必要があります。整数は「int型」、小数点がある場合は「float型」、文字は「str型」というラベルを貼って渡すことで、コンピュータが正しく処理できるようになるのです。

演算子 意味 例
+ 足し算 5 + 2 → 7
- 引き算 5 - 2 → 3
* 掛け算 5 * 2 → 10
/ 割り算 5 / 2 → 2.5
// 割り算（整数のみ） 5 // 2 → 2
% 余り 5 % 2 → 1
** べき乗 5 ** 2 → 25

また、print() 関数はカッコ内の内容を表示し、type() 関数は何型かを表示します。# を使うと、プログラムに影響しない「コメント」を残すことができます。

2.2 補足で解説

補足として、僕自身も最初誤解していた「関数」について解説します。

基本的に関数は 関数名() と書きますが、関数名はあくまで「何をするか」が書かれた「レシピ」です。そこに () がつくことで初めて、そのレシピを使って料理を作る（＝プログラムを実行する）ことができるようになります。

つまり、カッコは関数に「動作」を吹き込むもの、というのが本質的な理解です。

また、変数の名前をつけるときは、サプーさんの動画のように result_1 など、一目で中身がわかる名前にするのがおすすめです。初心者のうちから読みやすいコードを意識していきましょう！

3. 今日のまとめ

今回の学習内容を簡単にまとめました。

  • ・Pythonは上から順に実行され、「インデント（空白4つ）」でコードをまとめる。
  • ・データには「型（int, float, str）」があり、種類によってコンピュータの振る舞いが変わる。
  • ・関数は「レシピ（名前）」と「実行スイッチ（カッコ）」の組み合わせで動く。
  • ・変数の名前は result_1 のように、中身がひと目でわかる名前をつける。
