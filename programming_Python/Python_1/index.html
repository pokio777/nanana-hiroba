1. はじめに

皆さんこんにちは。「ナナナの広場^_^」管理人のpです。今回から、Pythonの勉強記録を記事として書いていこうと思います。この記事を特に読んでほしい方は、以下の通りです。

  • ・Pythonの勉強をこれから始める初学者の方
  • ・Pythonの勉強をしているけれど、上手く継続できない方

皆さんとこのブログを通して、一緒にPythonの勉強ができたら幸いです。実際、僕もPython初心者ですが、この記事を書きながら自分のスキルを高めていこうと思っています。

2. -1- Pythonの勉強計画について

これからどのように学習を進めていくか、僕なりの考えをまとめました。

2.1 何故0円なのか

それはズバリ「楽しく勉強していくため」です。僕の経験上、いろいろ教材を買って気を張り詰めて勉強する時、何かテストなどがあれば頑張れますが、やはりテストがない勉強は苦しくて途中で挫折してしまうことがあります。今回は肩の力を抜き、楽しく勉強をするために、あえて教材を買わずに0円で気軽に挑戦することに決めました。

2.2 具体的な今回の勉強計画

教材やスクールは一切利用せず、「YouTubeの講義動画」と「図書館の参考書」を活用します。

まずは「サプーさん」のPython速習動画で一通り学んだ後、「Python100本ノック」に毎日5本取り組み、前日の5本を復習することにします。学んだ知識に、初学者ならではの疑問を混ぜ合わせた解説をこの記事に記載していく予定です。

出典：YouTube「Pythonプログラミング VTuber サプー」

また、モチベーション維持のために「プログラマーやすなりさん」の動画を参考に、Pythonでマリオを作成しようと思います。

出典：YouTube「プログラマーやすなり」

単なる座学だけでなく、実際にゲームを作りながら学んでいくことで、Pythonのコーディングの癖やライブラリの使い方を楽しく身につけていければと考えています。
