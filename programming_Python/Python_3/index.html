1. はじめに

皆さんこんにちは。「ナナナの広場の管理人」のpです。今回も「Python100本ノック」の解説をしていきます。この記事を特に読んでほしい方は、Pythonの勉強をこれから始める初学者の方や、上手く継続できない方です。一緒にスキルを高めていきましょう。

2. Pythonノック6から10の解説

学習の中で重要だと感じたコードの書き方や関数の使い方をまとめました。

2.1 問題に出題された知識の確認

・長いコードを見やすくするコツ
コード全体を `()` で囲んで演算子の前で改行するか、もしくは `\`（バックスラッシュ）を挟んで演算子の前で改行します。こうすることで、長いコードを二行に分けることができ、非常に見やすくなります。

・数値を丸めるround関数
`round()` 関数は小数点以下を丸めることができ、括弧の中に丸めたい値を入れます。Pythonでは「端数0.5」の扱いが独特ですので、後ほど詳しく解説します。

・コードを短く書く「拡張代入演算子」
変数に値を代入する際、`x = x + 100` を `x += 100` といったように簡略化して表記できるのが拡張代入演算子です。

・文字を置き換えるreplace関数
`文字列.replace("置換前の文字列", "置換後の文字列")` という書き方で特定の文字を置き換えます。文字を消去したい場合は、置換後の文字列を `""`（空文字）にしてください。

2.2 補足で解説

・端数の処理と「銀行丸め」について
Pythonの `round()` では「銀行丸め（偶数への丸め）」が採用されています。端数がちょうど「5」の場合、結果が偶数になる方へ合わせる処理です（例：1250の十の位を丸めると1200になります）。第2引数で指定する桁数は、一の位を基準に小数方向は正の数、十の位方向は負の数で指定します。

・文字列フォーマットの種類
動画で使われていた `f"文字列{変数}文字列"` という「f文字列」は、直接変数を埋め込めるため最も推奨されます。他には `+` を使って結合する方法もありますが、数値を `str()` で変換する手間がかかります。

3. 今日のまとめ

今回は、コードを読みやすくする改行のコツ、Python特有の「銀行丸め」を行うround関数、記述を簡略化できる拡張代入演算子、そして便利なreplace関数やf文字列について解説しました。基礎的な内容ですが、効率的なコーディングには欠かせない知識です。次回も一歩ずつ進んでいきましょう！
